# Profile Analysis Workflow (Annual Monitoring)

This document records the workflow and menu logic for the Profile Analysis step of the Annual Monitoring Report Analyses in ProfCalc.

---

## Profile Analysis Menu Structure

1. **Extract & Prepare Shoreline Data**
    - Extract shoreline positions from survey profiles
    - Convert 2D to 3D (XYZ + date)
    - Import and process design template shoreline positions
2. **Calculate Shoreline Metrics**
    - Calculate distances (baseline to shoreline, design template to survey)
    - Generate H-1 and H-2 report tables (with highlights for landward positions)
3. **Summarize & Review Results**
    - Create summary table (H-3: min, max, avg, percent seaward)
    - View/plot summary statistics
4. **Export Results**
    - Export tables (CSV, Excel, shapefiles)
    - Export plots/figures
5. **Back to Annual Monitoring Menu**

---

*This structure is designed to streamline the annual shoreline analysis process, grouping related actions for clarity and efficiency. Each menu item corresponds to a logical step in the workflow, minimizing manual intervention and supporting robust, repeatable reporting.*
